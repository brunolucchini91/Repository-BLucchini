# Repository-BLucchini

Funcionalidades:

Registrar cuenta: Permite la creación de nuevos usuarios mediante un formulario que solicita:
Nombre
Apellido.
Mail.
Contraseña.
Reiteración de contraseña.
País de nacimiento.

Iniciar Sesión (Login): Acceso directo para usuarios ya registrados. Pide Usuario y contraseña.

Recuperar Contraseña: Opción para restablecer la contraseña en caso de olvido. Pide mail, nueva contraseña y reiteración de la nueva contraseña.

Validación de credenciales: Verifica que el correo electrónico y la contraseña ingresados coincidan con los datos almacenados para permitir el acceso al sistema. Pide correo electrónico y contraseña.

Verificación de usuario existente: Comprueba si el usuario se encuentra registrado antes de permitir el inicio de sesión. Pide correo electrónico
