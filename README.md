# Repository-BLucchini

Funcionalidades:

Registrar cuenta: Permite la creación de nuevos usuarios mediante un formulario que solicita:
Nombre
Apellido.
Mail.
Contraseña.
Reiteración de contraseña.
País de nacimiento.

Iniciar Sesión (Login): Acceso directo para usuarios ya registrados. Pide Usuario y contraseña.

Recuperar Contraseña: Opción para restablecer la contraseña en caso de olvido. Pide mail, nueva contraseña y reiteración de la nueva contraseña.
